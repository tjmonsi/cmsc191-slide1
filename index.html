CMSC 191 -GH2

Introduction to HCI

Created by TJ Monserrat

Course Description

A course on introducing human-computer interaction methodologies and design thinking.

At the end of this course...

Apply the principles of design thinking in solving human problems

Explain what Human-Computer Interaction is and how it impacts the world.

Be able to create a working prototype as a solution to a given problem.

Exercises

For each project, you'll have 10 exercises

  1. Interview
  2. Storyboard and Persona
  3. Brainstorm 1
  4. Brainstorm 2
  5. Dirty Prototype
  6. Testing Dirty Prototype
  7. Create Working Prototype
  8. Testing Working Prototype
  9. Create Video Prototype
  10. Testing Video Prototype with Working Prototype

Grading System

  • 1.0 - Did the 2 projects, submitted 2 videos, did all exercises, did 2 design thinking crash courses

Grading System - Subtractions

  • If did only one design thinking crash course: -0.25
  • If did not do any design thinking crash course: -0.5
  • If did only 70% of exercises: -0.25
  • If did only 50% of exercises: -0.5
  • If did not do any of the exercises: -1.0

Grading System - Subtractions

  • Only submitted 1 video: -0.5
  • Didn't submit any video: -1.0
  • Submitted 1 project: -1.0
  • Didn't submit any project: -2.0

Grading System - Notes to live by...

  • If after the grade subtraction, grade is < 3.0 then grade is INC
